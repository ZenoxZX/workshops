# Technical Workshops

A collection of interactive workshops covering modern development tools and techniques.

## Workshops

| Workshop | Description |
|----------|-------------|
| [Git Workflow](./git-workflow/) | Branching strategies, commit conventions, and collaborative workflows |
| [GPU Pipeline](./gpu-pipeline/) | GPU architecture, rendering pipelines, and optimization techniques |
| [Claude Code](./claude-code/) | AI-assisted development with Claude Code CLI |

## Getting Started

Visit the [live site](https://USERNAME.github.io/workshops/) or clone locally:

```bash
git clone https://github.com/USERNAME/workshops.git
cd workshops
# Open index.html in your browser
```
